## 📝 Daily Check-in 打卡流程

每天早晚（或学习后）在 `daily/` 下创建或更新当天的笔记文件：

1. **今日学习** — 读了什么、做了什么实验
2. **理解与收获** — 关键概念和自己的理解
3. **问题与阻碍** — 卡在哪里、需要什么帮助
4. **下一步** — 计划明天做什么
5. **进度标记** — ✅ 完成 / 🔄 进行中 / ❌ 阻塞

## 📬 Handbook Feedback 流程

在 `handbook-feedback/` 下提交反馈，格式见模板：

- 勘误与修正
- 补充案例或图解建议
- 新增知识节点提议
- 代码示例改进

提交前先查 [GitHub Issues](https://github.com/lxdao-official/aiweb3school/issues) 确保不重复。
