## 提交规则

1. **先查重** — 确认 [GitHub Issues](https://github.com/lxdao-official/aiweb3school/issues) 上没人提过
2. **新增文件** — 每条反馈一个 `.md` 文件，命名：`topic-description.md`
3. **包含内容**：
   - 章节路径（e.g., `handbook/bridge/agent-wallet/`）
   - 问题描述（错误 / 缺失 / 不清楚）
   - 建议修改方案
   - 参考资料（如果有）
